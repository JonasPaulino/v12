# V12 ERP

## Estrutura

- `web/`: aplicação web do ERP e seus serviços atuais (`front`, `back`, `acbr`, `payments`, `message`, `reports`).
- `desktop/`: área reservada para o futuro V12 PDV/Servidor Local/Estações.
- `docker-compose.yml`: compose da aplicação web, mantido na raiz nesta etapa para preservar o deploy atual.
- `deploy.sh`: script de deploy da aplicação web, mantido na raiz nesta etapa para preservar o fluxo atual.
